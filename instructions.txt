```javascript
// manifest.json
{
  "manifest_version": 3,
  "name": "ChatGPT Redirect",
  "version": "1.0",
  "description": "Redirects new tabs to ChatGPT",
  "permissions": ["tabs"],
  "background": {
    "service_worker": "background.js"
  }
}

// background.js
chrome.tabs.onCreated.addListener((tab) => {
  if (tab.pendingUrl === "chrome://newtab/") {
    chrome.tabs.update(tab.id, { url: "https://chat.openai.com" });
  }
});

```

2. `background.js`: This is a service worker that runs in the background. It listens for the creation of new tabs and redirects them to ChatGPT if they're empty new tabs.
